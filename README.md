# Vue3.X
Vue3.X+Element

#### 卸载vue2.x
````
npm uninstall vue-cli -g
````
#### 安装vue3.x
````
npm install -g @vue/cli
````
#### 更换淘宝镜像源
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
````
#### 创建vue3.x项目
````
vue ui
````

#### 安装依赖
````
npm install
````
#### 运行项目
````
npm run serve
````
